# Инструкция по использованию PNG-шаблона билета

## Как использовать

1. **Создайте PNG-файл шаблона билета** с именем `ticket_template.png`
2. **Поместите файл** в корневую папку проекта (рядом с `index.js`)
3. **Настройте координаты** в коде `index.js` под ваш шаблон

## Текущие координаты (можно настроить)

В файле `index.js` найдите секцию с координатами и отрегулируйте значения:

```javascript
const routeValueY = canvasHeight * 0.22;   // Y-координата для Tag Route и Tag Plate
const timeValueY = canvasHeight * 0.42;    // Y-координата для даты и времени
const checkCodeValueY = canvasHeight * 0.62; // Y-координата для кода проверки
const qrCenterY = canvasHeight * 0.75;     // Y-координата центра QR-кода
```

## Что бот подставляет на шаблон

1. **Tag Route** - номер маршрута (например, "03E") - слева от центра
2. **Tag Plate** - код маршрута (например, "980BK09") - справа от центра
3. **Дата и время** - в формате "dd.MM.yyyy HH:mm" - по центру
4. **Код проверки** - крупным шрифтом - по центру
5. **QR-код** - по центру внизу

## Если шаблон не найден

Если файл `ticket_template.png` отсутствует, бот создаст базовый белый canvas и будет работать как раньше (для обратной совместимости).
